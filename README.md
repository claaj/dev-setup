# Dev setup

This repo contains a my `neovim` and `tmux` config  that I use for development.

Dependencies:
- `neovim`
- `tmux`
- `perl`
- `npm`
- `cargo`
- `make`
- `ninja`
- `meson`
- `stow`
- `fzf`

## Install apps

~~~
cargo install starship --locked
~~~

~~~
cargo install tmux-sessionizer --locked
~~~
